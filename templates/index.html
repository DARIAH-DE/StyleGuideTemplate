Purpose

The purpose of the DARIAH-DE StyleGuideTemplate (SGT from now on) is to provide a basic template for all web applications that are part of the DARIAH-DE ecosystem.

Tip: If you are about to start developing a new DARIAH-DE application, by following the SGT the look & feel of your app is going to be consistent with the rest of the applications.
Tip: If you have already developed a DARIAH-DE application, you may want to adjust its layout so as to match the one described by the SGT.

Dependencies

This template does use external resources

    Fonts
  • Museo Sans Rounded: See here
  • Roboto: See here

Note, that Themify icons are defined as SVG symbols at the end of this html file. To extend with additional files, checkout the corresponding website, e.g. the svg-definition of the alert.

The design of the SGT uses SCSS which is compiled using the definitions in the gulpfile. If you want to change aspects of the design, you have to proceed as follows.

Setup

If not already availaible, install node.js, which comes with npm. Then, install all dependencies with npm i --dev.

You also need to install gulp globally with npm i -g gulp.

Compile and watch

Compile SCSS to CSS by running gulp styles. Compile JavaScript with gulp scripts. Run gulp compile to execute the tasks above.

The default task, executed with just gulp, runs the compile task and watches for changes.

How to use this template?

Clone the github repository and start adapting the template to your own application (e.g. integration with the template engine of your choice).

What to change

  • the tool name
  • the "breadcrumbs" section
  • the list of supported languages
    Tip: To find the language list look for this element in the HTML source code: 
    <ul id="language_selection">...</ul>
  • the licence information
    Tip: Look for this element in the HTML source code: 
    <div id="footer">...</div>

What not to change ( WORK IN PROGRESS )

The DARIAH-DE Global Menu must remain unchanged (i.e. the dropdown menu in the top-left corner which is re-populated from a global JSON definition).

Content

Templates

The /templates/ directory contains the following template variants:

  • front-page.html: A template for the front page in German.
  • front-page-en.html : A template for the front page in English.
  • html-elements.html: A list of possible HTML elements.
  • login.html: A template for a login page.
  • news-post.html: A new post template.
  • select-organisation.html : A template for the organisation selection.
  • status.html: A template for the status page of the app.
  • full-width.html: A template using the full-width of the browser window.

Resources

The assets and content directories contain all the files required by the SGT in order to display correctly.

The assets directory contains fonts, logos, the favicon, scripts and styles, whereas the content directory contains images.

If you move the assets or content/ directory make sure you change the path of links pointing to its content (e.g. js, css, icons, etc.).
